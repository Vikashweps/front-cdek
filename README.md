# 🎄 Тайный Санта СДЭК

Корпоративное веб-приложение для новогодней игры «Тайный Санта» с полным циклом управления участниками, вишлистами и распределением подарков.

## Описание проекта

**Тайный Санта СДЭК** — это интерактивная платформа для проведения корпоративной игры по обмену подарками. Приложение обеспечивает полный цикл взаимодействия: от регистрации участников до анонимного чата между Сантой и получателем.


### Ключевые возможности

- 🔐 **Регистрация и авторизация** — создание профиля участника с персональными данными
- 📝 **Управление вишлистом** — добавление, редактирование и просмотр желаемых подарков
- 🎲 **Автоматическое распределение** — система случайного назначения получателей
- 💬 **Секретный чат** — анонимное общение между участниками
- 🎨 **Адаптивный дизайн** — корректное отображение на всех устройствах
- 🎄 **Новогодняя тематика** — праздничный UI в корпоративных цветах СДЭК

## Технологии

| Технология | Назначение |
|------------|-----------|
| **React** | Библиотека для построения UI |
| **Vite** | Сборка и dev-сервер |
| **React Router DOM** | Клиентская маршрутизация |
| **CSS3** | Стилизация без препроцессоров (модульные стили рядом с компонентами) |act - Библиотека для построения пользовательского интерфейса 

### Почему Vite?

- **Мгновенный запуск** — сервер разработки стартует за миллисекунды
- **Hot Module Replacement** — мгновенное обновление без потери состояния
- **Оптимизированная сборка** — esbuild + Rollup для production
- **Минимальная конфигурация** — работает из коробки для React

##  Установка и запуск

### Предварительные требования

- npm или yarn

  
###  Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Vikashweps/front-cdek.git

# 2. Перейти в папку проекта
cd front-cdek

# 3. Установить зависимости
npm install
```
###  Запуск
```bash
# Запустить сервер разработки
npm run dev
```
После запуска приложение будет доступно:
http://localhost:5173/
